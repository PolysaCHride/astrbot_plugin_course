# astrbot_plugin_course

AstrBot 课表插件：绑定个人课表后，可输出**今日课表 / 明日课表 / 本周课表 / 下周课表**，并可配置**每日自动推送**和**开课前提醒**（提前时间可自定义）。

## 环境要求

- AstrBot：建议 **>= 4.19.4**
- 插件依赖：见 `requirements.txt`
- 安装了 playwright 环境，用于渲染课表。

## 安装

1. 在Astrbot控制台中选择Astrbot插件→右下角加号→从文件安装，选择本插件进行安装，依赖会自动安装;或下载源代码zip，手动将本插件目录放到 AstrBot 的插件目录，在插件目录执行 `pip install -r requirements.txt`安装依赖。
2. 在 AstrBot WebUI 中启用/重载插件。

## 指令

### 课表管理

- `/绑定课表`：进入绑定流程（120秒内发送 `.ics` 文件；发送"退出"可取消）
- `/删除课表`：删除已绑定的个人课表

### 课表查询

- `/今日课表`：输出今日课表图片
- `/明日课表`：输出明日课表图片
- `/本周课表`：输出本周课表图片
- `/下周课表`：输出下周课表图片

### 配置管理

- `/设置每日推送`：配置每日课表自动推送（回复"开启 HH:MM"如"开启 07:00"，或回复"关闭"禁用）
- `/设置提醒时间`：配置开课前提醒的提前时间（回复分钟数，范围 1-120）
- `/查看设置`：查看当前配置

## 绑定说明

上传 `.ics` 文件：在 120 秒会话窗口内直接发送 `.ics` 文件（在 wake up 课程表中获取）。

## 每日推送

- 使用 `/设置每日推送` 开启后，插件会在每天指定时间自动推送当日课表图片。
- 推送时间格式：HH:MM（24小时制，例如：07:00、18:30）
- 每天只推送一次。

## 开课提醒

- 插件会每 60 秒检查一次即将开始的课程。
- 默认提前 15 分钟提醒，可通过 `/设置提醒时间` 自定义（1-120 分钟）。
- 命中后会主动发送提醒消息，内容包含课程名与地点（若地点为空则仅发送课程名）。

## 数据存储

为避免插件更新/重装覆盖数据，课表与绑定信息会存储在 AstrBot 数据目录下：

- `data/plugin_data/astrbot_plugin_course/bindings.json`
- `data/plugin_data/astrbot_plugin_course/ics/*.ics`

## 常见问题

- 渲染图片失败：确认 AstrBot 安装了 Playwright 环境，必要时查看 AstrBot 日志中的报错信息。

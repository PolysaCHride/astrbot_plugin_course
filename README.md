# astrbot_plugin_course

AstrBot 课表插件：绑定个人课表后，可输出**今日课表 / 明日课表 / 本周课表**（图片），并可配置**每日自动推送**和**开课前提醒**（提前时间可自定义）。

## 环境要求

- AstrBot：建议 **>= 4.9.2**（使用插件数据目录与会话控制/文转图能力）
- 插件依赖：见 `requirements.txt`

## 安装

1. 将本插件目录放到 AstrBot 的插件目录（例如：`AstrBot/data/plugins/astrbot_plugin_course/`）。
2. 安装依赖：在插件目录执行 `pip install -r requirements.txt`（或通过 AstrBot 插件安装流程自动安装）。
3. 在 AstrBot WebUI 中启用/重载插件。

## 指令

### 课表管理
- `/绑定课表`：进入绑定流程（60 秒内发送 `.ics` 文件，或发送 WakeUp 分享口令；发送"退出"可取消）
- `/删除课表`：删除已绑定的个人课表

### 课表查询
- `/今日课表`：输出今日课表图片
- `/明日课表`：输出明日课表图片
- `/本周课表`：输出本周课表图片
- `/下周课表`：输出下周课表图片

### 配置管理
- `/设置每日推送`：配置每日课表自动推送（回复"开启 HH:MM"如"开启 07:00"，或回复"关闭"禁用）
- `/设置提醒时间`：配置开课前提醒的提前时间（回复分钟数，范围 1-120）
- `/查看设置`：查看当前配置

## 绑定说明

绑定流程支持两种输入：

1. **上传 `.ics` 文件**：在 60 秒会话窗口内直接发送 `.ics` 文件。
2. **WakeUp 分享口令**：发送包含 `「32位口令」` 的文本（插件会从中提取口令并拉取课表数据）。

绑定成功后，插件会记录当前会话的 `unified_msg_origin`，用于后续主动提醒发送到同一会话。

## 每日推送

- 使用 `/设置每日推送` 开启后，插件会在每天指定时间自动推送当日课表图片。
- 推送时间格式：HH:MM（24小时制，例如：07:00、18:30）
- 每天只推送一次，避免重复。

## 开课提醒

- 插件会每 60 秒检查一次即将开始的课程。
- 默认提前 15 分钟提醒，可通过 `/设置提醒时间` 自定义（1-120 分钟）。
- 命中后会主动发送提醒消息，内容包含课程名与地点（若地点为空则仅发送课程名）。

## 数据存储

为避免插件更新/重装覆盖数据，课表与绑定信息会存储在 AstrBot 数据目录下：

- `data/plugin_data/astrbot_plugin_course/bindings.json`
- `data/plugin_data/astrbot_plugin_course/ics/*.ics`

## 常见问题

- 发送了 `.ics` 但没识别：请确认在绑定会话的 60 秒内发送，且平台适配器的文件消息能被 AstrBot 转换为 File 组件并提供可下载 URL。
- 渲染图片失败：确认 AstrBot 的文转图能力可用（Playwright 环境正常），必要时查看 AstrBot 日志中的报错信息。
- 每日推送未生效：请检查 `/查看设置` 确认推送已开启且时间正确，注意使用 24 小时制。
